# Movie Recommender System

## Overview
A content-based movie recommendation system built using Python and Scikit-Learn.

## Features
- Recommends similar movies
- Uses CountVectorizer for feature extraction
- Cosine similarity for recommendations

## Dataset

This project uses the TMDB 5000 Movies Dataset.

Download:
https://www.kaggle.com/datasets/tmdb/tmdb-movie-metadata


## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-Learn
- Jupyter Notebook
